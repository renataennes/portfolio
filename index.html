Previsão de custos de Seguro de Saúde

Este projeto utilizou Ciência de Dados para investigar o impacto de variáveis biométricas e hábitos de vida, como o tabagismo, nos custos de seguros de saúde. Através de uma análise exploratória rigorosa (EDA), identifiquei correlações críticas que fundamentaram a construção de um modelo preditivo capaz de estimar gastos hospitalares com alta precisão.

O resultado foi uma solução que permite à operadora de saúde prever sinistros e segmentar clientes de forma estratégica, utilizando técnicas de regressão para transformar dados brutos em previsões financeiras confiáveis.

Fintech Data Insights

Neste projeto, apliquei um olhar minucioso sobre grandes volumes de dados transacionais para identificar padrões de comportamento e gargalos operacionais. O foco foi a construção de uma estrutura que unisse o processamento de dados via SQL e Python à clareza visual, permitindo o acompanhamento em tempo real da saúde financeira do negócio

O resultado final foi um ecossistema de monitoramento que automatiza a extração de dados e oferece dashboards interativos, facilitando a detecção de anomalias e o suporte rápido à tomada de decisão estratégica.
